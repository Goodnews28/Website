Runner - Snail Jump

Python


Description:

The Runner-Snail-Jump game is a Python-based 2D endless runner game where the player controls a character running forward, dodging obstacles by jumping.


Game Features:

  • Endless Runner Gameplay: The core mechanic of the game is endless running, meaning the snail continuously moves forward in the game world. The player has no control over the speed or direction but must focus on jumping to avoid obstacles that appear randomly.
  • Increasing Diffculty: As the game progresses, the speed at which obstacles appear increases, making it harder to react in time. This ramping difficulty keeps the player engaged and adds excitement to the gameplay.

Utilization:

  • Pygame for Game Mechanics
  • Sprite Animation
  • Game Loop
  • Collision Detection
  • Jump Mechanics
  • Sound Integration
